# rails-angularjs-app
This is an AngularJS's message board demonstrative application.

##Utilis:
This project uses Angular Front-end(with Sinatra server), with ES6 and Ruby on Rails Back-end.

# Dependencies:
- Ruby
	- Rails
- Node.js
	- NPM
	- Bower
	- Gulp
	- Babel

# Getting started:
```sh
	npm run start
	gulp dev
```
